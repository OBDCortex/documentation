Optimized Team Structure & Leadership

Structured roles for embedded system development using Yocto Linux on Raspberry Pi, ensuring clarity and accountability in documentation, integration, build, and communication.

Embedded Linux Engineer
Project Manager & Supervisor • Documentation oversight
Leadership Role: Project Manager
Key Mission

"Establish the Yocto-based OS and build pipeline, ensuring all deliverables are documented and reproducible."

Main Responsibilities
  • Board support (BSP), kernel, and device-tree integration
  • Automate Yocto builds (CI/CD)
  • Provide SDK/images for embedded team
  • Review and maintain project documentation
Cross Collaboration
  • Supports and reviews documentation for all technical roles
  • Coordinates image updates with integration and middleware engineers
Embedded Software Engineer
Cloud Integration & Communication Lead
Leadership Role: Cloud & API Integration
Key Mission

"Transform hardware data into actionable intelligence and maintain reliable cloud/api interfaces."

Main Responsibilities
  • Design and implement CAN bus data parsing and processing
  • Develop core daemons and API protocol (REST/MQTT)
  • Maintain communication bridge with mobile/cloud services
  • Robust logging and service management (systemd)
Cross Collaboration
  • Works with hardware and mobile engineers to test, refine, and document API endpoints
Hardware Integration Engineer
Documentation Lead
Leadership Role: Documentation
Key Mission

"Bridge physical interfaces and digital logic, and maintain main technical documentation for hardware/software integration."

Main Responsibilities
  • Enable/validate CAN/Ethernet hardware (device-tree overlays, pinmux)
  • Develop/maintain HAL libraries (SocketCAN API)
  • Write test scripts for hardware validation
  • Owns and updates hardware+integration documents
Cross Collaboration
  • Works directly with Linux and software engineers to ensure all hardware features are correctly integrated and documented
Mobile Application Developer
No formal leadership role
Key Mission

"Deliver a user-friendly mobile interface for diagnostics, configuration, and visualization."

Main Responsibilities
  • Develop and maintain Android/iOS UI app
  • Implement networking client to communicate with device API
  • Build app-based tools for configuration, discovery, and data display
  • Provide consumer documentation/user guides as needed
Cross Collaboration
  • Tests API and provides feedback to Embedded Software Engineer
