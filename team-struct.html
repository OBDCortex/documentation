Team Structure & Leadership

Defined roles for embedded Linux development, integration, and documentation to deliver a robust automotive diagnostics system.

Embedded Linux Engineer

Project Manager & Supervisor • Documentation Oversight
Project Manager

Key Mission

"Establish the Yocto-based OS and build pipeline, ensuring all deliverables are documented and reproducible."

Main Responsibilities

  • Board support (BSP), kernel, and device-tree integration
  • Automate Yocto builds and manage CI/CD pipelines
  • Provide SDK and system images to embedded developers
  • Review and maintain comprehensive project documentation

Cross Collaboration

  • Supports documentation and kernel-related needs of other technical roles
  • Coordinates build and image updates with Embedded Software and Hardware engineers

Embedded Software Engineer

Cloud Integration & Communication Lead
Lead: Cloud & API Integration

Key Mission

"Transform hardware data into actionable intelligence and maintain reliable cloud and API interfaces."

Main Responsibilities

  • Develop CAN bus data parsing and diagnostic processing
  • Create core daemons and implement API protocols (REST, MQTT)
  • Maintain communication bridge with mobile and cloud services
  • Implement robust logging and service management (systemd)

Cross Collaboration

  • Works with Hardware Integration and Mobile teams to test and document APIs

Hardware Integration Engineer

Documentation Lead
Documentation Lead

Key Mission

"Bridge physical interfaces and digital logic, and maintain main technical documentation for hardware/software integration."

Main Responsibilities

  • Configure CAN/Ethernet hardware (device-tree overlays, pinmux)
  • Develop and maintain HAL libraries (e.g. SocketCAN API in C/C++)
  • Create test scripts for hardware validation and diagnostics
  • Own and update hardware and integration documentation

Cross Collaboration

  • Coordinates closely with Embedded Linux Engineer and Software Engineer

Mobile Application Developer

No Formal Leadership Role

Key Mission

"Deliver a user-friendly mobile interface for diagnostics, configuration, and visualization."

Main Responsibilities

  • Develop Android/iOS app UI using modern frameworks (Jetpack Compose, Flutter)
  • Implement networking client communicating with device APIs
  • Build app tools for configuration, device discovery, and real-time data display
  • Create user-facing help guides and API consumer documentation

Cross Collaboration

  • Tests APIs and provides feedback to Embedded Software Engineer
